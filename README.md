# 🎓 Proyecto SoyUCAB 

Este repositorio contiene el backend (Node.js/Express) y la maquetación inicial del frontend para la red social universitaria SoyUCAB. Utiliza PostgreSQL como base de datos.

## 📋 Requisitos Previos

Antes de iniciar, asegúrate de tener instalado:

1.  **Node.js**: (v16 o superior).
2.  **PostgreSQL**: Base de datos instalada y corriendo.
3.  **pgAdmin 4**: (Opcional, pero recomendado para gestionar la BD).
4.  **Git**: Para clonar el repositorio.

## 🚀 Instalación y Configuración

Sigue estos pasos en orden para poner en marcha el proyecto:

### 1. Clonar el repositorio
Descarga el proyecto en tu máquina local:
```bash
git clone [https://github.com/mendozita01/ProyectoBD.git](https://github.com/mendozita01/ProyectoBD.git)

cd mi-red-social-backend
npm run dev
